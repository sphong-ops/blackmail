Each of 8 vertices colored R/B with prob 1/2. Want P(∃ rotation k: B+k ⊆ R), where B = blue set, R = red set.

Equivalent: ∃ k ∈ {0,...,7} with (B+k) ∩ B = ∅.

By |B|:
|B|=0: 1 coloring, works.
|B|≥5: impossible (|B|+|B+k| > 8).
|B|=1: 8 colorings work.
|B|=2: all 28 work (≤2 forbidden k of 7).
|B|=3: all 56 work (difference set has ≤6 nonzero values, so some k works).
|B|=4: need B+k = R (complement) for some k≠0. Enumerate:
  - k=1: alternating, 2 sets ({0,2,4,6}, {1,3,5,7}).
  - k=2: each 4-cycle alternates, 4 sets ({0,1,4,5}, {0,3,4,7}, {1,2,5,6}, {2,3,6,7}).
  - k=4: choose one from each pair {i,i+4}, 16 sets.
  - k=3,5,7: equal to k=1 set (alternation forced).
  - k=6: equal to k=2 set.
  - A_1, A_2, A_4 pairwise disjoint. Total 22.

Total favorable: 1+8+28+56+22 = 115. Probability 115/256, gcd=1. m+n = 371.
